### Your Personal AI Care Taker

BAYMAX is an intelligent healthcare web platform designed to simplify access to medical services. The system integrates **doctor recommendation, appointment booking, pharmacy assistance, health guidance, and emergency support** into a single platform.

The goal of this system is to **reduce waiting time, improve healthcare accessibility, and provide intelligent health assistance** through a modern web-based healthcare application.

---

# 📌 Project Overview

Accessing healthcare services can often be difficult due to:

* Limited medical awareness
* Difficulty in finding suitable doctors
* Long waiting times in hospitals
* Scattered healthcare resources

**BAYMAX** solves these problems by providing a unified healthcare platform where users can:

* 🔎 Find specialized doctors based on symptoms
* 📅 Book appointments online
* 💊 Check nearby pharmacies and medicine availability
* 🧾 Access personal health records
* 🤖 Receive AI-based health guidance
* 🚨 Get emergency assistance through SOS features

The system improves healthcare accessibility and provides convenient digital medical services.

---

# 🌐 Existing System

One of the popular healthcare platforms currently available is **Apollo 24|7**, which provides online doctor consultations, medicine ordering, and healthcare services.

🔗 Website: https://www.apollo247.com/

However, most existing platforms mainly focus on **consultation and appointment booking**.

**BAYMAX enhances this concept by introducing:**

* AI-based illness assistance
* Real-time doctor waiting count
* Pharmacy price comparison
* Medicine reminders
* Integrated emergency support system

# ⚙️ Key Features

### 👨‍⚕️ Doctor Recommendation

Users can enter symptoms or illness and the system recommends suitable doctors based on specialization.

### 📅 Online Appointment Booking

Patients can book doctor appointments directly through the platform.

### 💊 Medicine & Pharmacy Assistance

Users can search medicines and view nearby pharmacies along with availability and price details.

### 🧾 Personal Health Records

The system stores appointment history, prescriptions, and medical reports for easy access.

### 🤖 AI Health Assistant

An AI chatbot provides basic health guidance and answers user health queries.

### 🧭 Navigation Support

Integrated map services help users navigate to nearby hospitals and pharmacies.

### ⏰ Medicine Reminder

Prescription-based reminders help users take medicines on time.

### 🚨 Emergency SOS

Emergency mode provides quick access to emergency contact numbers and nearby hospitals.

---

# 🏗️ System Architecture

The project follows a **MERN Stack Architecture**.

**Technology Flow:**

User → Frontend (React) → Backend API (Node.js + Express) → MongoDB Database

This architecture ensures:

* Scalability
* Better maintainability
* Separation of frontend and backend logic

# 🔧 Installation & Setup

## 1️⃣ Clone the Repository

```
git clone https://github.com/VickyBoy2309/BAYMAX.git
cd BAYMAX
```

---

## 2️⃣ Backend Setup

```
cd "Source Code/BackEnd"
npm install
```

Create a `.env` file inside **BackEnd**:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
NODE_ENV=development
JWT_SECRET=your_secret_key
```

Run the backend server:

```
node server.js
```

Server will run on:

```
http://localhost:5000
```

---

## 3️⃣ Frontend Setup

Open another terminal:

```
cd "Source Code/FrontEnd"
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```
